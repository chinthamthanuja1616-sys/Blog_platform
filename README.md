# Blog Platform with Comments

## Overview
This is a Flask-based Blog Platform where users can register, log in, create blog posts, edit or delete their posts, and comment on blog posts. The project uses SQLite as the database and Bootstrap for the user interface.

## Features
- User Registration
- User Login & Logout
- Password Authentication
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- View All Blog Posts
- Add Comments
- SQLite Database Integration
- Responsive Bootstrap Interface

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite
- HTML
- CSS
- Bootstrap

## How to Run
1. Install the required packages:
   ```
   pip install -r requirements.txt
   ```
2. Run the application:
   ```
   python app.py
   ```
3. Open your browser and visit:
   ```
   http://127.0.0.1:5000
   ```
